# DEFERRALS

Track explicit deferred issues that were raised by internal/external reviews.

## Entry Format

```markdown
## DEFER-{N}

- Date: YYYY-MM-DD
- Phase raised: N
- Target phase: N
- Severity: MAJOR | MINOR | NIT
- Source: INTERNAL | EXTERNAL
- Summary: <what was deferred>
- Rationale: <why deferred>
- Risk: <impact of deferral>
- Owner: <person/team>
- Status: OPEN | CLOSED
```
